# Agent Status Check

Run this command to verify all agents are loaded:

```
@agents-registry status
```

Or ask Cursor:
```
Show me all available agents and their status
```

## Quick Agent Test

Test each agent category:

**C-Suite:**
- `@fractional-ceo-agent status`
- `@fractional-cfo-agent status`
- `@fractional-coo-agent status`

**Development:**
- `@principal-dev-agent status`
- `@dev-sync-agent status`

**Project Management:**
- `@git-project-manager-agent status`
- `@project-manager-agent status`
- `@program-manager-agent status`

**Operations:**
- `@change-management-agent healthcheck`
- `@infrastructure-agent status`

**Specialized:**
- `@historian-agent record`
- `@link-validation-agent status`
