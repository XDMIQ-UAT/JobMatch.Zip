# Cursor Agents Configuration

This file ensures agents are available in every Cursor chat session.

## Agent Availability

All agents defined in `.cursor/agents/` are automatically available in Cursor chats.

### How to Use Agents

Simply mention the agent name with `@` prefix:

```
@fractional-ceo-agent strategic_review
@principal-dev-agent review
@git-project-manager-agent scan_repos
```

### Agent Status Indicator

When an agent is invoked, Cursor will:
1. Load the agent configuration from `.cursor/agents/{agent-name}.json`
2. Execute the requested command or provide agent capabilities
3. Show agent context and available commands

### Available Agents

See `agents-registry.json` for the complete list of 15 available agents.

### Verification

To verify agents are loaded, ask Cursor:
- "List all available agents"
- "Show agent registry"
- "@agents-registry status"
